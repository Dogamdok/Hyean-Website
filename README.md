# Hyean Website Project

Imweb 기반 `hyean.org`를 자체 운영 가능한 최신 웹 스택으로 재구축하기 위한 프로젝트입니다.

## 목표
- 프론트엔드/백엔드 직접 관리 가능한 구조로 전환
- 기존 혜안 포트폴리오 및 핵심 메시지 이관
- `로컬콘텐츠 중점대학`/`RISE` 담당자 타깃 전환 구조 내장

## 범위 고정
- 본 저장소는 **(주)혜안 기업 웹사이트** 구축/운영만 다룹니다.
- `Hyean Heritage`는 별도 사이트로 분리하여 별도 프로젝트에서 구축합니다.

## 프로젝트 구조
- `apps/hyean-web`: Next.js 웹앱(프론트 + API 라우트)
- `docs`: 재구축/이관/전략 문서
- `content/migration`: 원본 콘텐츠 분석 메모
- `scripts`: 이관 보조 스크립트

## 빠른 시작
```bash
cd apps/hyean-web
npm install
npm run dev
```
