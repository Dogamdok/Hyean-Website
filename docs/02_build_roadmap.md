# Build Roadmap

## Scope Lock
- 현재 구축 대상은 **(주)혜안 기업 웹사이트**로 한정한다.
- `Hyean Heritage`는 별도 사이트로 분리 구축한다.

## Phase 1: Foundation (완료)
- 프로젝트 폴더 생성
- Next.js 앱 기본 구조 구축
- 회사/서비스/프로젝트/인사이트 데이터 구조 정의
- 문의 API(`POST /api/inquiries`) 구현

## Phase 2: Content Migration (진행 중)
- 핵심 프로젝트 본문/실행/지표 2차 이관 완료
- 인사이트 본문(요약+본문 단락) 2차 이관 완료
- 이미지/미디어 자산 선별 이관 진행 필요

## Phase 3: Conversion Optimization
- 타깃별 CTA A/B 테스트
- 문의 폼 후속 자동응답(메일/CRM) 연결
- 네이버/구글 검색 유입 리포트 대시보드 연동

## Phase 4: Deployment & Operations
- 배포 인프라 확정 (Vercel/Firebase 등)
- 도메인 전환/리다이렉트 전략 적용
- 운영 매뉴얼 및 월간 업데이트 루틴 정착
