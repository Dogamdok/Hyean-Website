# Project Scope

## 현재 범위
- 본 프로젝트는 **(주)혜안 기업 웹사이트** 재구축을 목표로 한다.
- 기존 Imweb 의존 구조를 벗어나, 직접 관리 가능한 코드베이스로 전환한다.

## 제외 범위 (현재 단계)
- `Hyean Heritage` 별도 브랜드/사이트는 본 프로젝트 이관 대상에서 제외.
- Heritage는 추후 별도 프로젝트로 분리 구축.

## 주요 목표
1. 기업 신뢰 자산(회사정보, 포트폴리오, 인사이트) 이관
2. 로컬콘텐츠 중점대학/RISE 담당자 전환 페이지 구축
3. 문의 수집 API 및 운영형 정보구조 구축
