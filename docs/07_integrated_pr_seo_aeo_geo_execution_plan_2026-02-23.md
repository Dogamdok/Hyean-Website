# HYEAN Integrated Growth Plan (PR + SEO + AEO + GEO)

> Date: 2026-02-23  
> Scope: Corporate website + owned media + search/discovery channels

## 1. Strategic Goal

혜안을 "지역 실행 프로젝트의 신뢰 가능한 운영 파트너"로 검색엔진과 생성형 AI 모두에서 일관되게 인식시키고,  
웹사이트를 문의 전환 중심 자산으로 고도화한다.

## 2. Positioning Framework

- Core Position: `설계-실행-성과관리`를 연결하는 현장형 파트너
- Proof Axis:
  1. 다기관 협업 정렬 능력
  2. 운영 로그/증빙 문서 기반의 성과관리
  3. 지역 현장 프로젝트의 확장 가능 모델 구축

## 3. Channel Architecture

1. Website (Owned)
   - 신뢰 증거 집적 허브
2. Search (SEO)
   - 문제/해결 의도 검색 유입 획득
3. Answer Engines (AEO)
   - 질문형 질의에 즉답 가능한 구조 확보
4. Generative Engines (GEO)
   - AI 요약/추천 맥락에서 인용 가능한 사실형 문장/지표 확보
5. PR/Media
   - 케이스 중심 외부 노출로 브랜드 검색 수요 확대

## 4. Execution Workstreams

## W1. Technical Foundation
- robots/sitemap 운영
- canonical/OG/twitter 메타 표준화
- JSON-LD 표준: Organization, WebSite, Article, BreadcrumbList

## W2. Evidence Content
- 사례 페이지 신뢰 블록 표준 템플릿
  - 기간/주체/역할/지표/출처/업데이트일
- 인사이트 페이지 출처 명시 및 주장-근거 연결

## W3. AEO/GEO Content Format
- FAQ + Q/A 블록
- 3~5개 핵심 수치 요약 블록
- 용어 정의 블록(지역실행/성과관리/다기관협업)
- 인용 가능한 1~2문장 단문 근거 문장

## W4. PR & Distribution
- 월 1회 사례형 보도자료(문제-개입-성과 구조)
- 분기 1회 심층 인터뷰/기고
- 캠페인형 외부 랜딩은 noindex 정책 유지

## W5. Measurement & Governance
- Search Console + Bing Webmaster + 분석 이벤트 통합
- 월간 점검 지표:
  - 비브랜드 유입, 상위 질의 CTR, 문의 시작/완료율
  - AI 인용 추적(브랜드+주제 질의 샘플링)

## 5. 8-Week Delivery Roadmap

## Week 1-2 (Foundation)
1. robots/sitemap/구조화데이터 배포
2. 홈 FAQ 및 핵심 페이지 메타 정비
3. 이미지/미디어 품질 이슈 정리

## Week 3-4 (Evidence Upgrade)
1. 대표 사례 6건 신뢰 블록 완성
2. 인사이트 4건 출처/근거 정비
3. 프로젝트별 canonical/OG 점검

## Week 5-6 (AEO/GEO Optimization)
1. 질문형 랜딩 섹션 확대
2. 문제-해결-지표 포맷 표준화
3. AI 검색 인용 문장 최적화

## Week 7-8 (PR + Conversion)
1. 사례 보도자료 배포
2. 문의 전환 UX 보강(초기 진단 CTA/후속 동선)
3. 성과 리뷰 및 다음 분기 백로그 확정

## 6. KPI Set

## Acquisition
- 비브랜드 검색 세션
- 상위 20개 타깃 질의 평균 순위/CTR

## Trust
- 사례 페이지 체류시간
- 출처 링크 클릭률
- FAQ 섹션 상호작용

## Conversion
- 문의 시작률
- 문의 완료율
- 문의→미팅 전환율

## AI Discovery (AEO/GEO)
- 브랜드+주제 질의에서의 언급/인용 여부
- AI 응답 내 핵심 수치/문장 정확 반영률

## 7. Immediate Next Actions

1. 기술 SEO 배포 상태 점검 (robots/sitemap/JSON-LD 색인 확인)
2. 대표 사례 6건의 출처/지표/업데이트일 입력
3. Search Console 색인 커버리지 + 쿼리 리포트 베이스라인 수집
