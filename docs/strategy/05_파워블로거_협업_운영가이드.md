# 파워블로거/상위블로거 협업 운영 가이드

## 1. 왜 필요한가
- 한국 검색 환경에서는 블로그/카페형 콘텐츠가 의사결정 초기 탐색 단계에 큰 영향을 준다.
- 특히 담당자 검색 시, "광고"보다 "실행사례를 설명한 포스팅"이 유리하다.

## 2. 협업 선정 기준
- 필수
  - 충남/충청권, 지역정책, 대학·공공사업 관심 독자 비중
  - 포스팅 품질(체류시간, 구조, 인용력)
  - 과거 협찬 콘텐츠의 신뢰도
- 제외
  - 트래픽만 높고 타깃 불일치
  - 과장형 카피/신뢰도 낮은 채널

## 3. 집행 포맷
- 포맷 A: 사례 분석형
  - 제목 예시: `충청권 대학이 찾는 지역활성화 실행 파트너의 조건`
- 포맷 B: 실무 인사이트형
  - 제목 예시: `RISE 사업 실무자가 확인해야 할 5가지 체크포인트`
- 포맷 C: 인터뷰형
  - 제목 예시: `부여 현장에서 본 청년정주형 콘텐츠 운영 모델`

## 4. 원고 구성 템플릿
1. 문제 정의(지역/사업 맥락)
2. 실행 접근(교육-실험-성과)
3. 사례 근거(검증 가능한 사실)
4. 담당자 관점의 의사결정 포인트
5. 문의 유도(랜딩/진단 신청)

## 5. 법적·윤리적 운영 원칙
- 협찬/광고성 콘텐츠는 관련 법령과 플랫폼 가이드에 맞게 명확히 고지한다.
- 허위·과장·비검증 수치 사용 금지.
- 사실 기반 문구만 사용하고, 필요 시 출처를 제시한다.

## 6. 운영 프로세스
1. 블로거 풀 구축(20~30개)
2. 샘플 3~5개 채널 테스트
3. 성과 상위 채널 재집행
4. 월별 카피/키워드/CTA 개선

## 7. KPI
- 포스팅별 클릭률
- 랜딩 유입수
- 문의 전환수
- 미팅 예약수
- 제안 요청수

## 8. 주의사항
- 단기 상위노출만 목표로 하면 신뢰 자산이 남지 않는다.
- "검색 상단 + 증빙 가능한 실력" 두 축을 같이 만들어야 계약 전환률이 오른다.
