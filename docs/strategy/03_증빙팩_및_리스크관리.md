# 증빙 팩 및 리스크 관리

## 1. 즉시 사용 가능한 증빙 문구(내부자료 기반)
- `주식회사 혜안` 명시 및 2025 청년마을 관련 공문 존재
- `2025년 청년마을 선정단체` 문구 포함 컨설팅 보고서 존재
- `충청남도 부여군` 주소/거점 정보 확인 가능
- 청년 대상 프로그램의 기수·인원·활동 항목 문서화

## 2. RISE 관련 활용 문구
- 외부 공개용 권장
  - `모 대학 RISE 연계 관광콘텐츠 교육 운영 사례 보유`
- 상세 수치(차시, 수료율, 지원금 등)
  - 외부 공개 전 사실 검증 후 반영
  - 필요 시 `내부 녹취/문서 기반`으로 출처 명시

## 3. 광고·홍보 리스크 관리
- 협찬형 블로그/유료 원고
  - 관련 법령 및 플랫폼 정책에 맞는 광고표시 준수
  - 허위·과장·기만 표현 금지
- 검색광고
  - 랜딩 도달 후 확인 가능한 사실만 광고 문구로 사용
- PR 기사
  - 인용·숫자는 증빙 문서와 1:1 매핑

## 4. 커뮤니케이션 원칙
- 금지
  - 검증되지 않은 성과 수치
  - 모호한 최상급 표현(무조건, 전국 최고 등)
- 권장
  - `수행 경험`, `사례 보유`, `검증 가능한 결과` 중심 표현
  - 수치 제시 시 파일 근거를 즉시 제시
