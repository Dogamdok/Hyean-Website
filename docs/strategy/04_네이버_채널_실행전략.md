# 네이버 채널 실행전략 (실무안)

## 1. 목적
- 검색 결과에서 "발견 가능성"을 높이고,
- 검색 후 검토 단계에서 "신뢰도"를 확보하며,
- 최종적으로 문의로 전환한다.

## 2. 채널별 역할
- 네이버 블로그
  - 실무형/사례형 콘텐츠 누적으로 키워드 점유
- 네이버 뉴스/포스트
  - 공신력 보강, 제3자 출처 확보
- 네이버 검색광고
  - 즉시 노출 확보, 키워드 반응 테스트
- 웹 랜딩페이지
  - 문의 전환

## 3. 콘텐츠 축(월 단위)
- 축 A: 수행 사례
  - 2025 청년마을 수행/고도화 과정
- 축 B: 방법론
  - 로컬콘텐츠·RISE 과제 설계 방법
- 축 C: 성과관리
  - KPI 설계, 증빙, 운영 리스크 관리
- 축 D: 지역성
  - 부여/충남/충청권 현장 기반 실행 스토리

## 4. 권장 운영 수치(초기 90일)
- 자사 블로그: 주 2건(총 24건)
- 협업 블로거: 월 4~8건
- PR 기사: 월 1~2건
- 검색광고: 소액 테스트 후 상위 키워드 증액

## 5. 성과 측정
- 노출: 키워드별 노출/클릭
- 신뢰: 브랜드 검색량, 제3자 언급량
- 전환: 문의 수, 미팅 예약 수
